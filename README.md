# 🎨 Frontend UI Showcase

A collection of modern, responsive, and interactive frontend applications built using **HTML**, **CSS**, and **JavaScript**. This repository demonstrates dynamic UI design, theme customization, DOM manipulation, and real-time user interactions through multiple mini projects.

---

## 🚀 Projects Included

### 📝 Interactive Blog Website

An engaging multi-category blog platform featuring interactive post functionality.

#### Features
- 🏠 Home page with interactive posts
- ❤️ Like button with dynamic like counter
- 💬 Add comments
- 💾 Save comments/posts
- 📤 Share functionality
- 💄 Cosmetics category
- 🐾 Animals category
- 🌙 Light & Dark mode
- 📱 Responsive interface

---

### ✅ Interactive To-Do Dashboard

A modern task management application with multiple UI themes.

#### Features
- ➕ Add new tasks
- ✔️ Mark tasks as completed
- 📋 View All, Pending, and Completed tasks
- 🗑️ Delete tasks
- 🧹 Clear completed tasks
- 📊 Remaining task counter
- 🎨 Three beautiful UI themes
- 📱 Responsive design

---

### 💰 Expense Tracker Dashboard

A clean finance dashboard for managing income and expenses.

#### Features
- 💵 Add income transactions
- 💸 Add expense transactions
- 📝 Transaction descriptions
- 📈 Automatic balance calculation
- 🟢 Income summary
- 🔴 Expense summary
- 📜 Transaction history
- 🌙 Dark Theme
- 💚 Green Theme
- 📱 Responsive interface
